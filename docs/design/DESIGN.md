# Riven Website Design Direction

## Design Read

Personal IP social-card website for social visitors, collaborators, and recruiters, with a luminous spatial persona language.

## Visual Principles

1. Riven is a kinetic identity, not a resume header.
2. IP avatar is the first visual anchor; real photo appears after the hero.
3. Rainbow is an information road only inside Rainbow Road.
4. Projects are interactive windows, not equal cards.
5. Work experience is a real-life dossier, not a table.
6. Contact is a signal console, not a footer.

## Palette

- Ink: `#101217`
- Muted ink: `#53606f`
- Ceramic: `#f9fbfb`
- Ice: `#e8f6fb`
- Signal blue: `#2f8cff`
- Glacier green: `#9ee6c9`
- Coral: `#ff8f70`
- Rainbow accents appear only inside route systems.

## Motion

- Hero: soft reveal and subtle avatar parallax.
- Project windows: hover or tap expansion.
- Rainbow Road: scroll-linked path growth.
- Work modal: morph/open transition.
- Contact: copy feedback pulse.
- Reduced motion: remove parallax and large translations.
